About

Hey everyone, as a football fan and developer wanted to combine both interests into this project. This project involves using a RESTful API service using FPL Player Data 2022-2023 and PL Teams' Stats and Results Data completed seasons. This will be done using Python Pandas for data cleaning and analysis. Backend API will be done using Express JS. I will also try to add some features that uses APIs to provide real time Match data such as upcoming fixtures, current season player statistics.
